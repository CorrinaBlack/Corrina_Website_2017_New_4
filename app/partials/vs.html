Visual Studio power for development needs

Visual Studio is an integrated development environment (IDE)
used to build Windows apps, websites, and web services. It
supports a variety of programming languages and
includes editors, visual designers, code navigation,
debugging, and code refactoring tools as well as tools
for the software develoment life cycle.


My role was Design Lead. I led design for editor and
code navigation, code refactoring, debugging, the
class designer, and source control. I also contributed
to the envisioning effort for Visual Studio Team
System which was targeted at large scale software
development life cycle support. My core
responsibilities included interaction design, visual
design, motion design, and art production.
